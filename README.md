# Instructions

  - Build the project:
    ```sh
    $ npm run install && npm run build
    ```

  - Start the project for a dev server (http://localhost:4200/):
    ```sh
    $ npm start
    ```

  - Execute the unit tests via [Karma](https://karma-runner.github.io)
    ```sh
    $ npm test
    ```
